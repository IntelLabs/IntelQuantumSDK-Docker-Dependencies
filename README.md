This repository is a read-only repository dedicated to archive [source code](/source_uris.txt) used in [IntelÂ® Quantum SDK v1.0 docker image](https://hub.docker.com/r/intellabs/intel_quantum_sdk/tags) to be compliant with the open source license obligations.

```bash
cat ${base_file}_part_* > ${base_file}
```
